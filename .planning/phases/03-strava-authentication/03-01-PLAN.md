## Phase Goal

(Direct transliteration of ROADMAP Phase 3 Goal "User authenticates with Strava; tokens managed securely" + the Phase 4/5 dependency chain; MVP mode, vertical-slice plan set: this plan locks every OAuth decision as a pure tested function, Wave 2 wires Android, Wave 3 makes it tappable, Wave 4 proves it live.)
